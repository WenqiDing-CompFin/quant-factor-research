# 量化研究实习周报模板

> 周报不是七篇日报的拼接。它需要总结本周最重要的研究判断、可复现证据、失败结果和下周优先级，让 Mentor 能据此决定继续、调整或停止某条研究路线。

## 第 [X] 周周报

```text
日期范围：YYYY-MM-DD 至 YYYY-MM-DD
本周研究主线：
仓库/分支：
本周数据性质：synthetic / 真实公开 / 其他
```

### 1. 本周研究目标

[保留 1-2 个可证伪目标，并写出事先约定的验收标准。]

```text
目标 A：验证 [因子/组合/数据规则] 是否在 [样本范围] 内表现出 [具体性质]。
验收标准：[Rank IC、ICIR、Q1-Q5 单调性或 spread、benchmark excess return、成本后结果、测试或数据质量阈值]。
```

|---|---|---|---|
| 数据处理 | [字段、日期、股票数、缺失/异常/发布日期规则] | [文件或表] | [完成/部分完成] |
| 因子研究 | [公式、方向、窗口、滞后、标准化] | [IC 文件或报告] | [完成/部分完成] |
| 组合研究 | [top fraction、benchmark、turnover、cost] | [收益文件或图] | [完成/部分完成] |
| 工程建设 | [模块、测试、配置、运行时间] | [commit/PR] | [完成/部分完成] |

### 3. 关键结果与研究判断

[只保留会影响研究决策的 3-5 条结果。每条包含数字、对照、结论和边界。]

```text
1. [指标] 为 [结果]，相对 [基线/benchmark/旧版本] 的差异为 [数值]；该结果 [支持/不支持/不足以判断] 原假设。
2. 成本从 [X] bps 提高至 [Y] bps 后，[指标] 从 [A] 变为 [B]，说明 [判断]。
3. 在 [子样本/参数/因子] 下结果 [保持/反转/消失]；尚未排除 [解释]。
4. 当前数据为 [synthetic/真实公开]，结论只能用于 [适用边界]。
```

### 5. 可复现性、偏差与风险检查

- 数据版本和来源是否记录：是 / 否
- 随机种子或实验配置是否记录：是 / 不适用 / 否
- `(date, ticker)` 唯一性、逐 ticker 月度连续性和价格有效性是否检查：是 / 否
- 信号时间、基本面可用时间、执行时间和收益时间是否区分：是 / 否
- 是否检查未来收益跨 ticker、缺失未来收益和退市处理：是 / 否 / 不适用
- benchmark 定义及其成本口径是否说明：是 / 否
- 是否报告 one-way turnover 和成本敏感性：是 / 否
- 是否存在前视、生存者、过拟合、多重检验或选择性汇报风险：[具体说明]
- synthetic 结果是否被误写成真实市场结论：否 / 需复核
- 保密检查是否完成：是 / 需复核
